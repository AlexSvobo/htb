## Converting to PDF

### Option 1: Using Pandoc (Recommended)
```bash
# Install pandoc if not already installed
sudo apt-get install pandoc texlive-latex-base texlive-fonts-recommended

# Convert markdown to PDF
pandoc Alex_Svobodny_Cybersecurity_Resume.md -o Alex_Svobodny_Resume.pdf --pdf-engine=pdflatex
```

### Option 2: Using Markdown Editors
- Open `Alex_Svobodny_Cybersecurity_Resume.md` in:
  - Typora (recommended)
  - Visual Studio Code with Markdown PDF extension
  - MacDown (macOS)
  - Mark Text
  - Or any markdown editor with PDF export

### Option 3: Online Tools
- Upload the markdown file to:
  - markdown2pdf.com
  - cloudconvert.com
  - dillinger.io (with export options)
