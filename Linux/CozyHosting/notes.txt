Synopsis
CozyHosting is an easy-difficulty Linux machine featuring a Spring Boot application with exposed Actuator endpoints. Enumeration leads to session hijacking, command injection, reverse shell, credential discovery, database access, password cracking, and privilege escalation via SSH misconfiguration.

- SSH Abuse Through Misconfiguration

Command Injection
