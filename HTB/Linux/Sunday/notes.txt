# HTB: Sunday (OSCP Study Notes)

**Machine Author:** Agent22  
**Prepared By:** Alexander Reid (Arrexel)  
**Difficulty:** Medium  
**Date:** 29th September 2018

---

## SYNOPSIS
- Focus: Finger service enumeration, weak credentials, sudo NOPASSWD exploitation.
- OS: Oracle Solaris 11.4.42.111.0 (Assembled December 2021)

---

## Enumeration

### Nmap
- Finger (port 79)
- SSH (port 22022)
- Other standard Solaris services

### Finger
- Enumerate users with `finger-user-enum` or Metasploit’s `finger_users` module.
- Use `/usr/share/seclists/Usernames/Names/names.txt` for a large username list.
- Discovered users: `sammy`, `sunny`, and many system accounts.

---

## Exploitation

### SSH Brute Force
- Hydra, Medusa, or Metasploit can be used.
- Credentials found:  
  - `sunny:sunday` (via Hydra and rockyou.txt)
- SSH access:  
  ```bash
  ssh -p 22022 sunny@10.10.10.76
  ```

---

## Privilege Escalation

### Backup Files
- `/backup/agent22.backup` and `/backup/shadow.backup` contain password hashes.
- Example hashes:
  ```
  sammy:$5$Ebkn8jlK$i6SSPa0.u7Gd.0oJOT4T421N2OvsfXqAT1vCoYUOigB:6445::::::
  sunny:$5$iRMbpnBv$Zh7s6D7ColnogCdiVE5Flz9vCZOMkUFxklRhhaShxv3:17636::::::
  ```
- Crack with John:
  ```bash
  john --wordlist=/usr/share/wordlists/rockyou.txt shadow.backup
  ```

### Sudo NOPASSWD
- As `sunny`, can run `/root/troll` as root:
  ```bash
  sudo -l
  sudo /root/troll
  ```
- `/root/troll` is a troll binary (does not grant root shell).

### Root via Sammy
- Crack `sammy`'s password from backup files.
- SSH as `sammy`.
- `sudo -l` as `sammy` reveals ability to run `sudo wget`.
- Overwrite `/root/troll` with a shell script using `sudo wget`.
- Race against a script that reverts `/root/troll` every second.
- Use two shells to overwrite and execute quickly for root shell.

- Finger user enumeration
- SSH brute-forcing
- Hash extraction and cracking
- Sudo NOPASSWD exploitation
- Race condition exploitation

## Useful Commands

```bash
# Enumerate users
finger-user-enum -U /usr/share/seclists/Usernames/Names/names.txt -t <target>

# SSH brute force
hydra -L users.txt -P /usr/share/wordlists/rockyou.txt ssh://<target> -s 22022

# Crack hashes
john --wordlist=/usr/share/wordlists/rockyou.txt shadow.backup

# Sudo privilege escalation
sudo -l
sudo /root/troll
```
